# Project 4 - Data Visualization

## Overview

This project focuses on visualizing an e-commerce dataset using Python and Matplotlib. Different charts were created to identify trends, patterns, and business insights from customer order data.

## Objectives

- Create meaningful visualizations from the dataset
- Analyze product sales distribution
- Visualize customer payment preferences
- Analyze order status distribution
- Study referral source performance
- Identify monthly sales trends
- Understand transaction value distribution

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset Information

- Dataset: Cleaned E-commerce Orders Dataset
- Total Records: 1200
- Source: Project 1 (Data Cleaning)

## Visualizations Created

### 1. Product Distribution (Bar Chart)
Shows the number of orders for each product category.

### 2. Payment Method Distribution (Pie Chart)
Displays customer payment preferences.

### 3. Order Status Distribution (Bar Chart)
Visualizes the distribution of different order statuses.

### 4. Referral Source Distribution (Bar Chart)
Shows which marketing channels generated the most customers.

### 5. Monthly Sales Trend (Line Chart)
Illustrates monthly sales performance over time.

### 6. Total Price Distribution (Histogram)
Shows the distribution of transaction values across all orders.

## Key Insights

- Printer is the most purchased product.
- Online is the most preferred payment method.
- Instagram is the leading referral source.
- Monthly sales fluctuate over time.
- June 2024 recorded the highest sales.
- Most customer orders fall within the low to medium price range.

## Conclusion

Data visualization was performed using Matplotlib to better understand customer purchasing behavior and business performance. Various charts highlighted important trends, sales patterns, payment preferences, referral effectiveness, and transaction value distribution. These visual insights support better business decision-making.
